# 公募基金托管法规知识库

## 核心用途

- 沉淀商业银行开展公募基金托管业务相关的法规原文、可检索文本、结构化元数据和业务专题地图。
- 支撑面向用户的法规咨询：回答问题时优先检索本地法规库，再按需联网核验现行性、最新修订和官方来源。
- 维护规则状态与依据链，明确区分现行有效、历史失效、待核验、待扩展和辅助资料，避免把征求意见稿、新闻稿、解读或行业手册当作现行强制依据。

## 知识库架构

- `00_入口与索引/`：知识库入口、HTML 总目录导航和更新记录。
- `01_法规原文库/`：按法规分类保存官方网页、PDF、附件等原始材料。
- `02_文本抽取库/`：保存从原文中抽取出的可检索文本，用于法规问答和条款定位。
- `03_元数据台账/`：维护规则编号、名称、状态、来源、路径、优先级和业务标签等结构化信息。
- `04_托管业务专题地图/`：按托管业务场景组织规则，作为法规咨询时的业务检索入口。
- `99_unresolved/`：记录待核验、待扩展、历史失效或来源不完整的事项。
- `tools/`：保存知识库构建脚本，用于下载、抽取、生成索引和刷新专题地图。

## 来源优先级与核验规则

1. 优先国家法律法规数据库、证监会、金融监管总局、AMAC、税务总局、交易所、中国结算等官方来源。
2. 正文页和附件均保留；栏目页、通知页、官方转载页只作为辅助索引或待核验线索。
3. 征求意见稿、起草说明、解读文章、新闻稿不进入现行主库；可进入辅助资料或 unresolved。
4. 每条规则保留 `current_status`、`validity_checked_at`、`source_type`、`business_tags` 字段。
5. 发现新旧版本冲突时，先标记待核验，不用草案替代正式现行规则。
